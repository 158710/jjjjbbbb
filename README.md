## FAQ(常见问题):
### Github部署日志:ETIMEOUT ERROR
ETIMEDOUT 错误腾讯是腾讯问题,初步怀疑同时部署人数较多
1. 切换是否开启境外加速(配置文件部署才支持)
2. 启用随机分钟部署(默认开启,在定时的小时随机抽取分钟进行执行).关闭请直接禁用Random Cron workflow
